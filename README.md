## Sobre o Projeto
Este repositório contém a entrega final da disciplina Projeto Aplicado I. O estudo foca na Análise Exploratória de Dados (AED) aplicada ao ciclo de vida de contratos da AET Consultoria Jurídica, com o objetivo de identificar gargalos e otimizar os fluxos de aprovação através de técnicas de Ciência de Dados e programação em Python.

## Arquivos do Repositório
* `Relatorio_Final_Projeto_Aplicado.pdf`: Documento completo com a apresentação da empresa, metadados, análise exploratória e proposta de solução.
* `trabalho.py`: Script em Python contendo a lógica de tratamento de dados (LGPD) e a geração de gráficos estatísticos (Pandas e Seaborn).
* `contratos_aet_oficial.csv`: Dataset anonimizado utilizado para o estudo.
